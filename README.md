# @amlplugins/groq-chat

Native `groq-sdk` SDK re-exported under the `@amlplugins` namespace.

> Groq Chat Completions — ultra-fast Llama/Mixtral/Gemma inference on Groq LPU, streaming + tool use.

## Install

```bash
npm install @amlplugins/groq-chat
```

## Use

```ts
import { /* named exports from groq-sdk */ } from "@amlplugins/groq-chat";
// or
import sdk from "@amlplugins/groq-chat";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
